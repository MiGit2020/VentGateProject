Here, a description of the data cadency

ECS: data point each minute, data structure: ecs, top, bottom, east ventage, west ventage position
weather data: Wind direction each minute, Wind speed each 5 min (hora ut), temperature in/out each 5 minutes 
WFS data: GMOS-OI/P2 each 30-45 seconds


PC: Datos de seeing establish en utc.  Chilean-time= UTC-3 en verano y UTC-4 en invierno.
